# ElixirScaffold

## Recommended: run with docker

The `./mix` scripts wraps mix calls in a docker:

```bash
# fetch dependencies
./mix deps.get
# to compile
./mix compile
# to test
./mix test
```

## Elixir dependencies

You can also [install Elixir](https://elixir-lang.org/install.html#unix-and-unix-like) on your machine.

Then:

```bash
# fetch dependencies
mix deps.get
# to compile
mix compile
# to test
mix test
```
